# VideoControls

> Adds some keyboard and mouse functionalities.

## Keyboard

- `f` - fullscreen
- `→` - skip 5 seconds forward
- `←` - skip 5 seconds backwards

## Mouse

- After being inactive for 2 seconds, the cursor will hide.

## Usage

```js
import VideoControls from '/path/to/folder/'

const videoControls = VideoControls(document.querySelector('video'))
```
